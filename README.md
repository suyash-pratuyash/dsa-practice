A structured daily practice repository for Data Structures and Algorithms in Java.

## Structure
- Arrays, Strings, Linked Lists, Trees, Graphs, Dynamic Programming, Sorting & Searching
- Each topic has its own directory with problems and solutions

## How to Navigate
- Problems are organized by topic
- Each solution includes:

    - Time & Space complexity
    - Multiple approaches when applicable
